# Change: Refactor Portal Homepage with Category Grouping and Visual Impact

## Why
当前主页存在以下问题：
1. **命名不统一** - 页面标题为英文（Video API Portal），卡片名称中英混杂
2. **功能分类不明确** - 6个入口卡片平铺展示，缺乏逻辑分组，用户难以快速定位
3. **图标缺乏视觉冲击力** - 所有图标都是单色线条 SVG，视觉吸引力不足
4. **卡片布局密集** - minmax(300px, 1fr) 导致卡片过小，空间感不足

## What Changes

### 1. 统一中文命名
- 页面标题：Video API Portal → AI 创作工作室
- 所有卡片名称和描述统一使用中文

### 2. 功能分类分组
将入口按用途分为三组：

**AI 生成**（创作类）
| 原名称 | 新名称 | 描述优化 |
|--------|--------|----------|
| 视频生成 | AI 视频 | Sora、Veo、Seedance 等模型 |
| 图像生成 | AI 图像 | 文字描述生成图像 |
| 电商图生成 | 电商图片 | 基于产品素材快速生成 |

**资产管理**（管理类）
| 原名称 | 新名称 | 描述优化 |
|--------|--------|----------|
| 产品库 | 产品素材库 | 产品图片识别与管理 |
| 存储库 | 生成记录 | 查看历史生成内容 |
| 图像处理 | 图像编辑 | 多图参考编辑 (Gemini) |

**系统设置**（仅管理员）
| 原名称 | 新名称 | 描述优化 |
|--------|--------|----------|
| Admin | 管理后台 | 用户管理与系统统计 |

### 3. 多彩渐变图标
每个功能使用独特的渐变配色：
- AI 视频：紫色→蓝色渐变
- AI 图像：蓝色→青色渐变
- 电商图片：橙色→粉色渐变
- 产品素材库：绿色→青色渐变
- 生成记录：蓝色→紫色渐变
- 图像编辑：粉色→紫色渐变
- 管理后台：灰色→蓝色渐变

### 4. 宽松大卡片布局
- 卡片最小宽度：300px → 360px
- 每行最多 2-3 个卡片
- 增加卡片内边距和图标尺寸
- 分组区域带标题标签

## Impact
- Affected code:
  - `frontend/src/pages/portal/portal-page.vue`
  - 可能新增 `frontend/src/styles/portal.css`
- No backend/API changes
- 保留全局 API 配置和 Admin 权限 gating 逻辑
